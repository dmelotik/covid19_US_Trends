How to download the data files

	1) go into covid19_US_Trends/application
	2) mkdir data
	3) cd data/
	4) git init
	5) git remote add origin https://github.com/CSSEGISandData/COVID-19
	6) git config core.sparseCheckout true # enable this
	7) echo "csse_covid_19_data/csse_covid_19_daily_reports_us" > .git/info/sparse-checkout
	8) git pull origin master

This will still download the whole repo, but condenses the files that aren't sparsed.

To update the local data repo

	1) go into covid19_US_Trends/application/data
	2) git pull origin master
